What is the difference between Margin, Border, and Padding?

If you’re new to CSS as I am you might have heard the words margin, border and padding but are not sure what they mean.

Once you understand the difference between margins, border and padding, you’ll be able to make better design decisions for your website. Let's have a look!


What is margin?

It is the space around an element's border that separtes elements between each other.

What is border?

In simple terms it is a line that separates elements between each other. It can be invisible or styled.

What is padding?

It is the space between the content and the border of any element, you can modify this as you want on CSS.
